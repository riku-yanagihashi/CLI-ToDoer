# Lazy Todo

## インストール方法

### MacOSの場合
RUSTの実行環境が整っている状態で以下のコマンドをターミナルで入力してください
```sh
git clone --depth 1  https://github.com/riku-yanagihashi/Lazy-todo.git ~/Lazy-todo && cd Lazy-todo/todo_cli && cargo build && cargo install --path . && cd 
```

※ Windowsはサポートする予定ありません

## アンインストール方法
```sh
cargo uninstall ltd
```
